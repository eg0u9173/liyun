# 第3节：下载、安装与订阅指南

> 时长：约18分钟 | 类型：录屏型 | 需要屏幕录制

---

## 开场白（30秒）

好，这节课我们进入实操环节。我会手把手带你完成 Codex 的下载、安装和订阅配置。全程录屏，你可以跟着一起做。

---

## Slide 2-3：下载桌面 App（3分钟）

【录屏演示】

首先打开浏览器，访问 openai.com/codex。页面上会有下载按钮，根据你的系统选择 Windows 或 macOS 版本。

下载完成后双击安装包，按照提示完成安装。安装过程很简单，基本就是下一步下一步。

安装完首次启动，它会要求你登录 OpenAI 账号。如果你还没有账号，需要先注册一个。

提示一下：确保你的网络环境能正常访问 OpenAI 的服务。

---

## Slide 4：订阅档位选择（3分钟）

登录后我们来看订阅。Codex 有三个档位。

Free 免费版，有使用次数限制，但足够你体验和入门。我建议先用免费版感受一下。

Plus 是 20 美金一个月，使用量明显增加，适合日常使用。如果你打算认真用，这个档位性价比最高。

Pro 是 200 美金一个月，最大使用量，适合重度用户或专业场景。

我的建议：先用 Free 版体验，确认 Codex 确实对你有帮助后再升级。

---

## Slide 5-6：第三方 API 接入（5分钟）

这个话题稍微进阶一点，但很重要，因为它可以帮你大幅降低成本。

为什么要用第三方 API 呢？因为 OpenAI 官方 API 按用量计费，用得多的话费用不低。而通过第三方 API，你可以用国产模型比如智谱 GLM，价格更便宜。

这里有三种接入方案。最简单的是手动修改 config.toml 配置文件。我来演示一下。

【录屏演示】

打开配置文件，位置在用户目录下的 .codex 文件夹里。你可以修改 provider 和 base_url 来指向第三方 API。

另外两种方案：Codex++ 是社区开发的图形化管理工具，CCX 加 CC Switch 是代理加一键切换的方案。这些适合进阶用户，新手暂时不需要。

---

## Slide 7：费用控制（2分钟）

有一点一定要做：设置月度消费上限。

在 OpenAI 后台的使用设置里，可以设置每个月的消费上限。这样即使你忘记关闭某些任务，也不会产生意外的巨额账单。

建议设置 10-20 美金的上限，日常使用完全够了。

---
