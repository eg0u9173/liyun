# 第15节：实战浏览器DKFile

> 详细讲解脚本

---

## 1. Chrome 控制（5分钟）

【录屏】启用浏览器能力，安装 Chrome 扩展。然后让 Codex 打开 B 站搜索教程视频，返回最佳结果链接。

---

## 2. 安全提醒（2分钟）

浏览器控制权限较高，从只读、低风险页面开始。不要操作支付、删除等高风险页面。

---

## 3. DKFile 发布（6分钟）

【录屏】注册 DKFile，手动上传一个 HTML 文件获得公网链接。然后演示 Codex 加 API 全自动发布。从设计网页到发布上线，全程自动化。

---
