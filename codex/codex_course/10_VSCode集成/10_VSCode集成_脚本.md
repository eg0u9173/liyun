# 第10节：VS Code 集成

> 详细讲解脚本

---

## 1. 安装与配置（3分钟）

【录屏】在 VS Code 扩展市场搜索 Codex 插件安装。认证后你会看到侧边栏多了 Codex 的入口。

---

## 2. 编辑器内使用（4分钟）

【录屏】选中一段代码，右键选择让 Codex 修改。或者在侧边栏直接对话。Codex 能理解你的项目结构，给出的建议很有针对性。

---

## 3. 代码审查与重构（3分钟）

【录屏】让 Codex 审查代码质量，它会给出具体建议。重构也一样，描述你想要的结构，Codex 帮你执行。

---


> 预计总时长：约10分钟

> 本节关键词：安装与配置, 编辑器内使用, 代码审查与重构
