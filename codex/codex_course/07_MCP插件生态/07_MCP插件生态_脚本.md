# 第7节：MCP 插件生态

> 详细讲解脚本

---

## 1. 什么是 MCP（3分钟）

MCP 全称 Model Context Protocol，是让 Codex 连接外部工具的标准协议。如果 Skills 是 Codex 的内置技能，那 MCP 就是它的外部接口，就像 USB 口一样，可以连接各种外设。

---

## 2. Draw.io MCP 实操（5分钟）

【录屏】安装 Draw.io MCP 后，你可以用自然语言画图。比如「画一个微服务架构图，包含网关、用户服务、订单服务和数据库」。几秒钟就生成了。点击链接可以查看和编辑，还能导出图片。

---

## 3. Playwright MCP（4分钟）

【录屏】Playwright MCP 让 Codex 能写和运行浏览器自动化测试。这对 Web 开发者特别有用。描述你要测试的页面和行为，Codex 自动生成测试脚本并运行。

---


> 预计总时长：约12分钟
